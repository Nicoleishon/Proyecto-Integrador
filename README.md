# Proyecto Integrador - Sistema de Gestión de Turnos Médicos

## Información Relevante

**Recepcionista:** Se puede iniciar sesión con el usuario `admin` y contraseña `1234`. Desde su panel puede registrar nuevos pacientes, asignar turnos, reprogramar o cancelar citas y consultar listados de pacientes y médicos.

**Paciente:** Puede registrarse desde el panel de inicio de sesión o ser creado por la recepcionista. Un ejemplo de un paciente ya registrado es el usuario `jperez` con contraseña `1234`. Una vez registrado, puede iniciar sesión y gestionar sus propios turnos, visualizando, solicitando o cancelando sus turnos.
